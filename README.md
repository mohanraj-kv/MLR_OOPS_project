🏠 House Price Prediction using Multiple Linear Regression (OOP)

📌 Overview
This project builds a House Price Prediction system using Multiple Linear Regression (MLR) implemented with an Object-Oriented Programming (OOP) approach.

It includes:
A training pipeline (main.py)
A saved machine learning model (Model.pkl)
A Flask-based web application (app.py) for real-time predictions

🎯 Objective
The goal of this project is to:

Predict house prices based on key features
Demonstrate OOP in machine learning
Build an end-to-end ML workflow (training → saving → deployment)
